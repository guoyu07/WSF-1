Macro Definition Documentation

#define DEFSNESECHO   "64"

'SNES' Echo value. WSFTool and WinWSFTool both feature a option to tag WSF files as 'SNES'-echo'd. This naming is simply a reference to how the results, if the track suits it, will sound like it is being played on an SNES. This macro simply defines how much echo we want to tag for. This is the equiv. of saying "SEcho=DEFNESECHO"
