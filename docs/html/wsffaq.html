FAQ (Frequently Asked Questions)
  • What is WSF?
    WSF is a format that packs a group of audio files (such as MODs) into a group of WSF files and a samplepack. This prevents the use of duplicate samples throughout a large library of music, such as a game soundtrack.
  • How can I start testing WSF?
    1. Download WSFPlay or INWSF if you have Winamp 2
    2. Download one of the sample WSF packs
    3. Load one of the WSF's into Winamp of WSFPlay and start headbanging
    4. If you experiance any troubles, contact Josh Whelchel
  • Will there be an MMF Extension?
    Actually, two! You can obtain both of these, ModFusion eX & ModFusion from either Syntesis or Apocalyptic Coders. However, it should be noted that ModFusion 1 (not eX) will only support WSF's made in compliance with the ModFusion 1 Specifications described in wsf.txt

  • Enter Password?
    If you're using WSFPlay or INWSF (and maybe other players have the feature) then you may be asked for a password when playing a WSF. Some WSF files are password protected so that not everyone can listen to them. If you enter an incorrect password, you will still be able to hear the song, but you will notice glitches and undesireable clicks in the playback. For more information on this see Protecting your Modules

  • Can I change filenames to include tag information?
    You may have many WSF/WTFs which have weird filenames, like SR_45.wtf etc. If you're a neat freak, you're going to want to name the wtfs so you can access the files quicker. To do this, all you have to do is use WSFTool's -N option.

    WSFTool *.wtf "-NSpanorak - %title%.wtf"


    This works just like INWSF's set title format option!
