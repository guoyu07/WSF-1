Enumeration Type Documentation

enum wsft_e

WSFTool Running Modes.

Enumerator:
WSFT_TOWSF 

Converts input to wsf -M.

WSFT_WSFTO 

Converts input FROM wsf -W.

WSFT_SHOWEXT 

Shows Extensions Avalible -showext.

WSFT_READPACK 

Reads a Pack and shows info -P.

WSFT_READPACKWSF 

Reads a WSF and shows its PACKs info (like READPACK) -R.

WSFT_MODTAGS 

Modifies Tags -T.

WSFT_MAKEWGM 

Creates a WGM from the selected files -G.

WSFT_LOADWGM 

Unpacks a WGM -U.

WSFT_NAME 

Renames WSF based on tags -N.

WSFT_REMPASS 

Removes passwords from WSF/WTF files -B.

WSFT_HELP 

Extra Help.

WSFT_PREVSAMP 

Previews Sample.

WSFT_PREVOGGSAMP 

Previews "Ogg'd" Sample.

WSFT_LISTSAMP 

Lists Samples in SampleID|SampleName|StoredSize|CFlag|CompSize format.

WSFT_COMPSAMP 

Compresses Sample.

WSFT_NONE 

Used when no mode is set.
