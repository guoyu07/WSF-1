enum wsfmodflag_e

This is the enumeration for the flags of samples in WSFs.

Enumerator:
WSFS_NORMAL 

Use original Sample.

WSFS_REVERSED 

Reverse the sample.

WSFS_SAMPCUTOFF 

Use the sample only until cutoff (p1=cutoff)

WSFS_SAMPAFTER 

Add end of sample after this one (p1=sampid)

WSFS_REVSAMPCUTOFF 

Use the sample only until cutoff (p1=cutoff) REVERSED.
